# Changelog

## [0.1.0] — Initial Release

### Added
- Syntax highlighting for all 20 Vasp DSL block types
- Code snippets for every block type with tab-stops
- Real-time diagnostics powered by the Vasp language server (Chevrotain)
- Context-aware completions for block keywords, entity names, page names, executor and provider enums
- Hover documentation for every keyword
- Go-to-definition for entity and page cross-references (workspace-wide)

# CLI Overview

The `vasp` CLI is your primary workflow tool.

```bash
vasp --version
```

## Typical workflow

1. `vasp new` to scaffold
2. Edit `main.vasp`
3. `vasp validate` and `vasp generate`
4. `vasp start` for development
5. `vasp build` and `vasp deploy` for production

::: tip
> Use `vasp add` commands for fast, incremental scaffolding.
